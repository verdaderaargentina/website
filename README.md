# Estancia La Verdadera Argentina — versión independiente v6

Esta versión replica la estructura, los textos, la disposición de imágenes, el mapa, la galería y el formulario de la página de Lovable, pero funciona con archivos propios.

## Verla
Abrí `index.html` en Safari o Chrome. Para probar el mapa, necesitás conexión a internet.

## Formulario
Envía consultas a `info@verdaderaargentina.com` mediante FormSubmit. El primer envío genera un correo de activación que hay que confirmar una sola vez.

## Publicar y modificar
Subí la carpeta a GitHub y conectala a Cloudflare Pages, Netlify o Vercel. Cada modificación futura se publica reemplazando los archivos o haciendo un nuevo commit; no hace falta volver a configurar el dominio.

## Google Workspace
Al conectar `verdaderaargentina.com`, no se deben borrar ni modificar los registros MX de Google Workspace.
